## What is this app?

A mobile-first web application for tracking blood pressure readings 3 times daily. Designed for simplicity and ease of use, especially for elderly users.

### Key Features

- **Simple BP Entry**: Large touch targets, auto-suggested time of day
- **Visual Dashboard**: See trends at a glance with color-coded status
- **History View**: Browse all readings with filters
- **Data Export**: PDF reports and CSV spreadsheets
- **Secure**: User authentication with Firebase
